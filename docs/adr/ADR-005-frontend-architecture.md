# ADR-005 - Frontend Architecture

## Status

Accepted

## Context

Se requiere una experiencia moderna y optimizada.

## Decision

Frontend:

- Next.js 16
- App Router
- TypeScript
- Server Components por defecto

## Consequences

### Positivas

- Excelente SEO.
- Buen rendimiento.

### Negativas

- Curva de aprendizaje.

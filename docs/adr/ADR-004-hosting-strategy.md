# ADR-004 - Hosting

## Status

Accepted

## Context

La plataforma necesita hosting simple y económico.

## Decision

Frontend:

- Vercel

Backend:

- Railway

Base de datos:

- Supabase

## Consequences

### Positivas

- Despliegues rápidos.
- Integración sencilla.

### Negativas

- Dependencia de varios proveedores.
